Made for the 2025 autumn lisp game jam
https://itch.io/jam/autumn-lisp-game-jam-2025

Instructions:

Survive as long as possible. Think pong mixed with asteroids. Try to get a high score. Stay tuned for the season pass update ;)

Keybinds:
- WASD to move
- Mouse to aim paddle
